💡 HOW IT WORKS:

1. User enters prompt in React UI
2. Frontend sends: POST /api/generate-reel {prompt, language}
3. Backend runs main.py in background thread
4. Sends real-time updates via Server-Sent Events:
   ✓ Parsing your prompt...
   ✓ Generating script...
   ✓ Finding video clips...
   ✓ Creating voice-over...
   ✓ Adding background music...
   ✓ Composing final reel...
5. Frontend shows progress with animations
6. Video ready → user can preview/download
7. File stored in: output/reel_*.mp4

🆘 QUICK TROUBLESHOOTING:

Q: API can't start / Port 5000 in use
A: netstat -ano | findstr :5000
   taskkill /PID <pid> /F

Q: Frontend won't connect to API
A: Check .env.local has VITE_API_URL=http://localhost:5000
   Check browser console (F12) for CORS errors
   Verify API server is running: http://localhost:5000/health

Q: npm not found
A: Install Node.js from https://nodejs.org/

Q: Very slow first run
A: HuggingFace model downloading (14.5GB)
   Subsequent runs use cached model
   Tip: Set HAS_HUGGINGFACE=False in main.py for instant tests

See INTEGRATION_GUIDE.md for more help!
